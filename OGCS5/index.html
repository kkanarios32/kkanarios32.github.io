    Cooperative Thread Array (CTA) [W2D5]

    • June 1, 2026

    NVIDIA's hardware-level term for what CUDA programmers call a thread block.

    A CTA is a group of threads that execute on the same Streaming Multiprocessor (SM), can synchronize via barriers (__syncthreads()), and share fast on-chip shared memory (Shared memory (GPU)). Threads within a CTA are scheduled and executed in groups of 32 called warps.
