Matrix Multiply Unit (MXU) [OGCRP]

  • May 22, 2026
  • Kellen Kanarios

A core component of the TPU architecture responsible for performing matrix multiplication via a systolic array.

  • 256x256 totaling 65,536 multiply accumulate units i.e. each unit can multiply and add in one cycle.
    • Remark. Can only do 256x256 matrix multiplication, so the order is
    1. Tiling the matrix to load from HBM to VMEM
    2. Compiler tiles from VMEM to 256x256 tiles to load into MXU.

    Vector Processing Unit [OGCRN]

    • May 22, 2026
    • Kellen Kanarios

    A core component of the TPU architecture responsible for elementwise operations.

    • 2D SIMD execution, where each of 1024 sites execute same program in lockstep.
    • Each site contains 4 ALUs, enabling 4096 FLOPs.
    • Fed a 8x128 Vector Register in one cycle.
