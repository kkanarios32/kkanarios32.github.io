          Normalizing Flows [B9J9]

          • July 15, 2025

          Brief notes on Variational Inference with Normalizing Flows to motivate flow matching.

          We are concerned with the problem of density estimation. Given a dataset \(\mathcal {X}\), consisting of samples \(\mathbf {x}_i \sim p(\mathbf {x})\), we want to learn \(p(\mathbf {x})\). One popular way to do this is via variational inference. As a brief recap, in variational inference, we assume that there is actually a joint distribution \(p(\mathbf {x}, \mathbf {z})\), where \(\mathbf {z}\) is a latent variable. We then find \(p(\mathbf {x})\) by maximizing the log-likelihood \(\log \int p(\mathbf {x} \mid \mathbf {z}) \mathrm {d}\mathbf {z}\)

          For an example of this, see my post on expectation maximization. In that post, we utilize the ELBO to learn a latent variable model \(q_{\phi }(\mathbf {z} | \mathbf {x})\). As discussed in my post on ELBO, in order to learn \(p(\mathbf {x})\), we must have that \(p(\mathbf {z} | \mathbf {x})\) is in the family \(q_{\phi }\). However, \(q_ {\phi }\) is usually a Gaussian parametrized by \(\phi \), so that we can easily learn \(\phi \). In this case, it is very unlikely that \(p(\mathbf {z} \mid x)\) lies in the family of \(q_{\phi }\) and therefore we have no hope of learning the true \(p(\mathbf {x})\).

          The probability density path \(p: [0,1] \times \mathbb {R}^d \to \mathbb {R}_{>0}\) is a time dependent probability density function i.e. \(\int p_t(x)\mathrm {d}x = 1\)

          Definition. Time-dependent vector field [008C]

          • July 1, 2025
          • Kellen Kanarios

          We define a time-dependent vector field as a map \(v: [0,1] \times \mathbb {R}^d \to \mathbb {R}^d\)

          Definition. Flow [008D]

          • July 1, 2025
          • Kellen Kanarios

          Give a vector field \(v_t\) one can construct a time-dependent diffeomorphic map called a flow \(\phi : [0,1] \times \mathbb {R}^d \to \mathbb {R}^d\) defined via the ordinary differential equation \[\begin {align*} \frac {\mathrm {d}}{\mathrm {d}t} \phi _t(x) &= v_t(\phi _t(x)) \\ \phi _{0}(x) &= x .\end {align*}\]

          Remark. Modeling a vector field via a neural network i.e. \(v_t(x; \theta )\) implicitly defines a deep parametric model of the flow \(\phi _t(x)\) via solving the ODE or an approximation, such as \[\phi _{t + \Delta t}(x) \approx \phi _t(x) + \Delta t \cdot v_t(\phi _t(x); \theta )\] Obviously, accuracy will be dependent on the step size \(\Delta t\).

          Definition. Generated probability paths [008E]

          • Kellen Kanarios

          We say that a vector field generates a probability density path if the flow of \(v_t\) satisfies \[p_t = [\phi _t]_* p_{0},\] where \[[\phi _t]_*p_{0}(x) = p_{0}(\phi _t^{-1}(x)) \det \left [ \frac {\partial \phi _t^{-1} }{\partial x}(x) \right ]. \]

          Theorem. Generation via Continuity Equation [008I]

          • July 2, 2025
          • Kellen Kanarios

          A vector field generates a probability density path if it satisfies the continuity equation.

          Definition. Flow Matching Loss [LRL4]

          • July 9, 2025
          • Kellen Kanarios

          Given an initial noise distribution \(p\), a desired data distribution \(q\), a probability density path \(p_t\), such that \(p_{0} = p\) and \(p_{1} =q\), and a vector field \(u_t\) that generates \(p_t\). We define the flow matching loss \(\mathcal {L}_{\mathrm {FM}}\) as \[\mathcal {L}_{\mathrm {FM}}(\theta ) \coloneqq \mathbb {E}_{t, p_t(x)} \|v_t(x; \theta ) - u_t\|\]

          Conditioning Trick FM [008F]

          • July 2, 2025
          • with contributions from Kellen Kanarios

          Given data distribution \(q(x)\), we define an approximation \(p(x)\) with the form \[p(x) = \int p(x \mid x_1) q(x_1) \mathrm {d}x_{1}\] In order to ensure \(p(x) \approx q(x)\), we pick \(p(x \mid x_{1})\) that concentrates around \(x_{1}\) i.e. \(p(x \mid x_{1}) \sim N(x \mid x_{1}, \sigma ^2 I)\) for small \(\sigma \).

          Definition. Conditional Vector Field [8XAJ]

          • July 9, 2025
          • Kellen Kanarios

          We define the conditional vector field \(v_t(\cdot \mid x_{1})\) as the vector field that generates the conditional probability path \(p_t(x \mid x_{1})\).

          Definition. Marginal Vector Field [373T]

          • July 9, 2025
          • Kellen Kanarios

          Given a conditional vector field \(v_t(x \mid x_{1})\) we define the marginal vector field as \[v_t(x) = \int v_t(x \mid x_1) \frac {p(x \mid x_1) q(x_{1})}{p(t)}\]

          Theorem. Marginal Generation [93R8]

          • July 9, 2025
