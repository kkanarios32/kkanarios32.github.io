Axiom. Axiom of Choice [001H]

  • November 26, 2024
  • Kellen Kanarios

The axiom of choice assumes the existence of a choice function \(f\), such that \[\forall X\left [\varnothing \not \in X\implies \exists f\colon X\to \bigcup _{A\in X}A\quad \forall A\in X\left (f(A)\in A\right )\right ]\,.\]

    Example. Vitali Sets [001G]

    • November 26, 2024
    • Kellen Kanarios

    If we consider \([0,1]\), then we can partition \([0, 1]\) into equivalent classes as \(x \sim y\) if \(x - y = r\) for some \(r \in \mathbb {Q} \cap [0, 1]\). Denote each of these as \(V_{r}\). By the Axiom of Choice, we can choose one representative from each \(V_r\). We call the set of these representatives \(V\). Therefore, we can write \([0, 1]\) as a countable union via \([0,1] = \bigcup _{r \in \mathbb {Q}} V + r\). For the Lebesgue measure \(\lambda \), each \(\lambda (V + r) = \lambda (V)\). Since each \(V + r\) is disjoint, we have \[\lambda \left (\bigcup _{r \in \mathbb {Q}} V + r\right ) = \sum _{r \in \mathbb {Q}} \lambda (V)\] If \(\lambda (V) = 0\), then \(\lambda ([0,1]) \leq \lambda (V) = 0\) which is a contradiction. Otherwise \(\lambda ([0,1]) = \sum _{r \in \mathbb {Q}} k = \infty \) for some \(k \in \mathbb {R}\). Thus, \(V\) is not Lebesgue measurable.
