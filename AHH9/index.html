3D-reconstruction is not actionable i.e. we can reconstruct a room with a cup but if you try to lift the cup the whole image moves (cup is attached to counter).

Drawer

  1. Anchor gaussians around the zero level-set
  2. Scene decomposition. Identify interactable and actionable objects
  3. Use LLM for reasoning over objects i.e. density, types of joints
  4. Recover invisible regions via differentiable rendering + generative modeling

2. Learning From Humans

  • September 22, 2025
  • Kellen Kanarios

Real-to-sim-to-real: record environment -> build simulation of environment -> train in simulator -> deploy in real world.

Ideas:

  1. Object state as an anchor
    • Get rid of human.
    • Find corresponding actions that match object states in trajectory.
  2. Still not robust for precise movements.
  3. Test-time alignment
    • Unroll in real world
    • Replay in simulation
    • Align

RL -> Imitation Learning

  1. Do imitation learning on noisy trajectories.
  2. Use noisy robot and human trajectories
  3. Use classifier to try to predict whether trajectory is human or robot
    • Noise here is the end noise in the diffusion training phase.
    • Only use indistinguishable trajectories.

Upshot is that they can "augment" robot data with human data in the diffusion training phase by including human trajectories when sampling high noise.
