Theorem. Behavior cloning error bound (with distribution shift) [OGCSA]

  • June 3, 2026

If \(\pi (a | s)\) is trained via ERM on \(s \sim d^{\pi _{\beta }}(s)\) and optimal labels \(a^*\), and attains generalizationerror \(\epsilon \) on \(s \sim d^{\pi _{\beta }}(s)\), then \(\ell (\pi ) \leq C + H^2 \epsilon \) is the best possible bound.
