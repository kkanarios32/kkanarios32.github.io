Definition. Consistent estimator [000F]

  • November 4, 2024
  • Kellen Kanarios

Formally speaking, an estimator \(T_{n}\) of parameter \(\theta \) is said to be weakly consistent. If it converges in probabilty to the true value of the parameter i.er \[\lim \limits _{n\rightarrow \infty }T_{n}=\theta . \quad \text { i.e. for all t>0 } \quad \lim \limits _{n\rightarrow \infty }\Pr \left (\left |T_{n}-\theta \right |>c\right )=0.\]
