    Theorem. Additivity of Intervals [000S]

    • November 20, 2024
    • Kellen Kanarios

    We have the following properties

    1. If \(\bigcup _{k}I_{k}\subset I\), and the \(I_{k}\) are disjoint, then \(\sum _{k}|I_{k}|\leq |I|\).
    2. If \(I\subset \bigcup _{k}I_{k}\) (the \(I_{k}\) need not be disjoint), then \(|I|\leq \sum _{k}|I_{k}|\).
    3. If \(I=\bigcup _{k}I_{k}\), and the \(I_{k}\) are disjoint, then \(|I|=\sum _{k}|I_{k}|\).

    Remark. The important takeaway from Additivity of Intervals is that extending (2) from finite to countable unions requires Heine-Borel theorem, illustrating that this extension is non-trivial. This will be important to remember as we continue to extend the Lebesgue measure to \(\sigma \)-algebras.
    This section aims to motivate why we need to approach probability from a measure theoretic point of view. As a concrete example, Dyadic Intervals sets up the dyadic-interval representation of an infinite coin toss, and Exercise 1.1 makes precise why no discrete probability space can carry that sequence.

    Example. Dyadic Intervals [000L]

    • November 14, 2024
    • Kellen Kanarios

    Represent coin toss as dyadic expansion consider the set

    \[\left [\omega :\ d_{i}(\omega )=u_{i},\,i=1,\ldots ,n \right ]=\Big (\sum _{i=1}^{n} \frac {u_{i}}{2^{i}},\ \sum _{i=1}^{n}\frac {u_{i}}{2^{i}} + \frac {1}{2^{n}}\Big ).\]

    More stuff

    I think my main takeaway from this is that infinite sequences break discrete probability because each discrete event will have probability zero. This is not actually a problem because in an infinite sequence the occurence of any such sequence "feasibly" has zero probability. However, we still want to reason about probabilistic things like the convergence of the average of the sequence. Since discrete probability is broken, we cannot even use it for these things. This is why we need to use measure theory to allow for zero probability sequences, but still provide information about these more "general" events. Who knows if this intuition is right though?

    Remark.
    • After a bit more digging, the main culprit here is the countable subadditivity condition of measure.
    • In a discrete probability space, the entire space is at most countable. Therefore, if each individual event has zero probability then the countable union i.e. the whole space will have zero probability.
    • In measure theoretic probability, we are able to accumulate non-zero probability mass from an uncountable union of zero measure events.
    • I guess the canonical example of this is that each point on a line has length 0, but the line itself (an uncountable union of points) has non-zero length.
    • In the dyadic intervals example, each event of an arbitrary discrete probability space can be identified with a point on the line of probability zero.

    Definition. \(\sigma \)-field [000O]

    • November 20, 2024
    • Kellen Kanarios

    A class of subsets \(\mathscr {F}\) of \(\Omega \) is a \(\sigma \)-field if the following holds

    1. \(\Omega \in \mathscr {F}\)
    2. If \(A \in \mathscr {F}\) then \(A^c \in \mathscr {F}\)
    3. If \(A_1, A_2, \ldots \in \mathscr {F}\) then \(\bigcup _{i = 1}^{\infty } A_i \in \mathscr {F}\)

    Definition. Probability Measure [000P]

    • November 20, 2024
    • Kellen Kanarios

    A set function is a real-valued function defined on some class of subsets of \(\Omega \). A set function \(P\) on a field \(\mathscr {F}\) is a probability measure if it satisfies these conditions:

    1. \(0\leq P(A)\leq 1\) for \(A\in \mathscr {F}\);
    2. \(P(\varnothing )=0\), \(P(\Omega )=1\);
    3. if \(A_{1},A_{2},\ldots \) is a disjoint sequence of \(\mathscr {F}\) sets and if \(\bigcup _{k=1}^{\infty }A_{k}\in \mathscr {F}\), then \[P\Big (\bigcup _{k=1}^{\infty }A_{k}\Big )=\sum _{k=1}^{\infty }P(A_{k}).\]

    Definition. Probability Space [000Q]

    • November 20, 2024
    • Kellen Kanarios

    If \(\mathscr {F}\) is a \(\sigma \)-field in \(\Omega \) and \(P\) is a probability measure on \(\mathscr {F}\), the triple \((\Omega ,\mathscr {F},P)\) is called a probability measure space, or simply a probability space. A support of \(P\) is any \(\mathscr {F}\)-set \(A\) for which \(P(A)=1\).

    Theorem. Continuity and countable subadditivity of a probability measure [000R]

    • November 20, 2024
    • Kellen Kanarios

    Let \(P\) be a probability measure on a field \(\mathscr {F}\).

    1. Continuity from below: If \(A_{n}\) and \(A\) lie in \(\mathscr {F}\) and \(\mathscr {A}_{n}\uparrow A\), then \(P(A_{n})\uparrow P(A)\).
    2. Continuity from above: If \(A_{n}\) and \(A\) lie in \(\mathscr {F}\) and \(A_{n}\downarrow A\), then \(P(A_{n})\downarrow P(A)\).
    3. Countable subadditivity: If \(A_{1},A_{2},\ldots \) and \(\bigcup _{k=1}^{\infty }A_{k}\) lie in \(\mathscr {F}\) (the \(A_{k}\) need not be disjoint), then \[P\left (\bigcup _{k=1}^{\infty }A_{k}\right )\leq \sum _{k=1}^{\infty }P(A_{k})\]

    Definition. Outer Measure [000T]

    • November 21, 2024
    • Kellen Kanarios

    For each subset \(A\) of \(\Omega \), \[P^{*}(A)=\mathrm {inf}\,\sum _{n}P(A_{n}),\] where \(A \subset \bigcup _{n = 1}^{\infty } A_n.\)

    Remark. Taking \(P_*(A) = 1 - P^*(A^c)\) gives the inner measure. This is just analogously approximating from the inside.
    Intuition is that the sets that are "measurable" should be sets that can be arbitrarily approximated from the inside and outside, or equivalently \[P^*(A) + P^*(A^c) = 1\]
    1. My intution for this is that if a set is not arbitrarily well approximated by countable other sets from our algebra then we cannot hope to get an accurate measure from our measure defined on only the algebra.
    2. The countableness is just from the definition of the \(\sigma \)-field.

    Then we can use \(P^*\) as our measure. To do this, we will need to show that \(P^*\) is countable additive as one might expect. The rest of the properties of a Probability Measure are trivial.

    Proof.

    Suppose that \(A \subset \bigcup _{n = 1}^{\infty } A_n\). By the definition of Outer Measure, we can find some \(B_{nk}\), such that \(A_n \subset \bigcup _{k = 1}^{\infty } B_{nk}\) where \[\sum _{k} P(B_{nk}) < P^*(A_n) + \epsilon 2^{-k}\] Now take \(C_n = \bigcup _{k} B_{nk}\), so that \(\bigcup _{n} A_n \subset \bigcup _{n} C_n\) and \[P^*\left (\bigcup _n A_n\right ) < \sum _{n} P(C_n) = \sum _{n} \sum _{k} P(B_{nk}) < \sum _{n} P^*(A_n) + \epsilon \]

    Since \(\bigcup _{n} A_n = A\), we have achieved the desired result.

    It turns out that for some reason only enforcing \(A\) and its complement to sum to \(1\) is not restrictive enough? From this, we end up with the final condition

    Definition. \(P^*\)-measurable [000U]

    • November 21, 2024
    • Kellen Kanarios

    \(A\) is \(P^*\)-measurable if for every \(E \subset \Omega \), we have that \[P^*(A \cap E) + P^*(A^c \cap E) = P^*(E)\]

    You then only need to show two things:

    1. The class of \(P^*\)-measurable sets contains \(\mathscr {F_0}\),
    2. The class of \(P^*\)-measurable sets is a \(\sigma \)-field.
    From these, we can conclude that \(P^*\) is at least defined on a \(\sigma \)-field containing \(\sigma (\mathscr {F}_0)\). Restricting \(P^*\) to \(\sigma (\mathscr {F}_0)\), then yields the desired extension.

    Uniqueness

    • October 30, 2024
    • Kellen Kanarios

    We now want to show that the extension we developed in the previous section is actually unique. The first question we have is what do we mean by unique? To answer this, we need to remember what we have done so far.

    1. We have a Probability Measure \(P\), defined on an algebra \(\mathscr {F}_0\).
    2. We then extended \(P\) to \(\sigma (\mathscr {F}_0)\) via \(P^*\), so that for every \(A \in \mathscr {F}_0\) we have \(P(A) = P^*(A)\) and \(P^*\) is defined on all of \(\sigma (\mathscr {F}_0)\)
    From this, our only restriction is that \(P^*(A) = P(A)\) for \(A \in \mathscr {F}_0\). Therefore, uniqueness here means that if \(P^*(A) = Q(A)\) for some \(Q\) and every \(A \in \mathscr {F}_0\). Then \(P^*(A) = Q(A)\) for every \(A \in \sigma (\mathscr {F}_0)\).

    Definition. \(\pi \)-system [0018]

    • November 26, 2024
    • Kellen Kanarios

    A class \(\mathcal {P}\) of subsets of \(\Omega \) is a \(\pi \)-system if it is closed under the formation of finite intersections: \[A,B\in \mathcal {P} \implies A\cap B\in \mathcal {P}.\]

    Definition. \(\lambda \)-system [0019]

    • November 26, 2024
    • Kellen Kanarios

    A class \(\mathscr {L}\) is a \(\lambda \)-system if it contains \(\Omega \) and is closed under the formation of complements and of finite and countable disjoint unions:

    1. \(\Omega \in \mathscr {L}\);
    2. \(A\in \mathscr {L}\) implies \(A^{c}\in \mathscr {L}\);
    3. \(A_{1},A_{2},\ldots ,\in \mathscr {L}\) and \(A_{n}\cap A_{m}=\varnothing \) for \(m\neq n\) imply \(\bigcup _{n}A_{n}\in \mathscr {L}\).

    I was a bit unsure that a \(\lambda \)-system is not a \(\sigma \)-field, but the author provided an example that I will walk through to understand the distinction

    Example. Consider a four point space \(\Omega = \{x_1, x_2, x_3, x_4\}\). We can define a \(\lambda \)-system by taking \(\mathscr {L} = \emptyset , \Omega ,\) and the six two point sets. The union of any two disjoint two point sets will be \(\Omega \). However, a non-disjoint union of two disjoint sets can result in a three point set, which is not in \(\mathscr {L}\). These non-disjoint unions would be covered in a \(\sigma \)-field.

    To remedy this disconnect, we have the following lemma:

    Lemma. A \(\pi \)-system that is also a \(\lambda \)-system is a \(\sigma \)-field [001A]

    • November 26, 2024
    • Kellen Kanarios

    A class that is both a \(\pi \)-system and a \(\lambda \)-system is a \(\sigma \)-field.

    In our example, we could take the intersection of say \(\{x_1, x_2\}\) and \(\{x_2, x_3\}\) to get \(\{x_2\}\). We could then take a disjoint union of \(\{x_1, x_3\}\) and \(\{x_2\}\) to get \(\{x_1, x_2, x_3\}\).

    Theorem. Dynkin's \(\pi \text {-}\lambda \) [001B]

    • November 26, 2024
    • Kellen Kanarios

    f \(\mathscr {P}\) is a \(\pi \)-system and \(\mathscr {L}\) is a \(\lambda \)-system, then \(\mathscr {P}\subset \mathscr {L}\) implies \(\sigma (\mathscr {P})\subset \mathscr {L}\).

    Now we will see how this allows us to prove our desired result i.e. the uniqueness of our extension of a probability on an algebra via the outer measure.

    Theorem. Uniqueness of Extension [001C]

    • November 26, 2024
    • Kellen Kanarios

    Suppose that \(P_{1}\) and \(P_{2}\) are probability measures on \(\sigma (\mathscr {P})\), where \(\mathscr {P}\) is a \(\pi \)-system. If \(P_{1}\) and \(P_{2}\) agree on \(\mathscr {P}\), then they agree on \(\sigma (\mathscr {P})\).

    Proof. Take \(\mathscr {L}\) to be the class of sets such that \(A \in \mathscr {L} \implies P_1(A) = P_2(A)\). Since \(\mathscr {P}\) is a \(\pi \)-system, from Theorem 9, it is sufficient to show that \(\mathscr {L}\) is a \(\lambda \)-system. Since \(\Omega \in \mathscr {P}\), \(\Omega \in \mathscr {L}\). If \(A \in \mathscr {L}\), then \[P_1(A^c) = 1 - P_1(A) = 1 - P_2(A) = P_2(A^c).\] Therefore, \(A^c \in \mathscr {L}\). Finally, if \(A_1, A_2, \ldots \) disjoint then \[ P_1\left (\bigcup _{n = 1} A_n\right ) = \sum _{n = 1} P_1(A_n) = \sum _{n = 1}P_2 (A_n) = P_2\left (\bigcup _{n = 1}A_n\right ) \] Thus, \(\mathscr {L}\) is a \(\lambda \)-system. From Theorem 9, \(\sigma (\mathscr {P}) \subset \mathscr {L}\) and \(P_1\) and \(P_2\) agree on all of \(\sigma (\mathscr {P})\).

    An (important?) note is that this works due to our definition of a Probability Measure. Namely, countable additivity is really a condition on countable disjoint unions, meaning that the \(\sigma \)-field induced by a Probability Measure does not make full use of the generality allowed under arbitrary countable unions because it must satisfy countable additivity.

    Example. Vitali Sets [001G]

    • November 26, 2024

    If we consider \([0,1]\), then we can partition \([0, 1]\) into equivalent classes as \(x \sim y\) if \(x - y = r\) for some \(r \in \mathbb {Q} \cap [0, 1]\). Denote each of these as \(V_{r}\). By the Axiom of Choice, we can choose one representative from each \(V_r\). We call the set of these representatives \(V\). Therefore, we can write \([0, 1]\) as a countable union via \([0,1] = \bigcup _{r \in \mathbb {Q}} V + r\). For the Lebesgue measure \(\lambda \), each \(\lambda (V + r) = \lambda (V)\). Since each \(V + r\) is disjoint, we have \[\lambda \left (\bigcup _{r \in \mathbb {Q}} V + r\right ) = \sum _{r \in \mathbb {Q}} \lambda (V)\] If \(\lambda (V) = 0\), then \(\lambda ([0,1]) \leq \lambda (V) = 0\) which is a contradiction. Otherwise \(\lambda ([0,1]) = \sum _{r \in \mathbb {Q}} k = \infty \) for some \(k \in \mathbb {R}\). Thus, \(V\) is not Lebesgue measurable.

    Theorem. Fatou's lemma for events [001K]

    • December 2, 2024
    • Kellen Kanarios

    For each sequence \(\{A_{n}\}\), \[P\left (\lim \inf _{n}A_{n}\right )\leq \lim _{n}\inf _{n}P\left (\,A_{n}\right ) \leq \lim _{n}\sup _{n}P\left (\,A_{n}\right )\leq P\left (\lim _{n}\sup _{n}A_{n}\right ).\]

    Proof. We define \(B_n \coloneqq \bigcup _{k = n}^{\infty } A_k\) and \(B_n\) as \(C_n = \bigcap _{k = n}^{\infty } A_k\). Note that \(\{B_n\}\) is a monotonically decreasing sequence to \(\lim \sup _n A_n\), and \(C_n\) monotonically increases to \(\lim \inf _{n} A_n\). Therefore, we can apply continuity from below / above such that \[ \begin {align*} \inf _n P(A_n) \geq P(C_n) \iff \lim _n \inf _n P(A_n) &\geq \lim _n P(C_n) \\ &= P(\lim _n C_n) \\ &= P(\lim \inf _n A_n) \end {align*} \] Similarly, \[ \begin {align*} \sup _n P(A_n) \leq P(B_n) \iff \lim _n \sup _n P(A_n) &\leq \lim _n P(B_n) \\ &= P(\lim _n B_n) \\ &= P(\lim \sup _n A_n) \end {align*} \]

    Definition. Independence [001O]

    • December 26, 2024
    • Kellen Kanarios

    Independence evolves in the following manner:

    1. Events \(A\) and \(B\) are independent if \(P(A \cap B) = P(A)P(B)\).
    2. A finite collection of events are independent if \[P(A_{k_1} \cap \cdots \cap A_{k_j}) = P(A_{k_1}) \cdots P(A_{k_j})\]
    3. A infinite collection of sets is defined to be independent if every finite subcollection is independent.
    4. Classes \(\mathscr {A}_1, \ldots , \mathscr {A}_n\) are independent if for any events \(A_1 \in \mathscr {A}_1, \ldots , A_n \in \mathscr {A}_n\), \(A_1, \ldots , A_n\) are independent.

    Theorem. Independent \(\pi \)-systems generate independent \(\sigma \)-algebras [001P]

    • December 27, 2024
    • Kellen Kanarios

    If \(\mathscr {A}_1, \ldots , \mathscr {A}_n\) are independent and each \(\mathscr {A}_i\) is a \(\pi \)-system then \(\sigma (\mathscr {A}_1), \ldots , \sigma (\mathscr {A}_n)\) are independent.

    Proof. We have sequence of independent classes and want to show that their respective \(\sigma \)-algebras are also independent. The idea is to take the class of events \(B_1\) that are independent with \(\mathscr {A}_2, \ldots , \mathscr {A}_n\), and show that \(\sigma (\mathscr {A}_1)\) is a subset of this class. To do this, we use Dynkin's \(\pi \text {-}\lambda \) by noticing that this class is a \(\lambda \)-system that contains the \(\pi \)-system \(\mathscr {A}_1\).

    Meaning in english. In the book, they give the example of even and odd numbered coin tosses. We want to know when a new event only depends on even or odd coin tosses. Concretely, we know that even and odd coin tosses are independent, and we want to know if, for example, the outcome of every fourth coin toss is independent of odd tosses. We can do this rigorously by showing that this event is in the \(\sigma \)-algebra of even coin tosses. Then by Theorem 5.7, we know that this event is also independent of odd coin tosses.

    Theorem. Borel-Cantelli One [001Q]

    • December 27, 2024
    • Kellen Kanarios

    If \(\sum _{n}P(A_n)\) converges, then \(P(\lim \sup _n A_n) = 0\).

    Proof. Recall that \(\lim \sup _n A_n \subset \bigcup _{k = n}^{\infty } A_k\). Therefore, \[P(\lim \sup _n A_n) \leq P\left (\bigcup _{k = n}^{\infty } A_k\right ) \leq \sum _{k = n}^{\infty } P(A_k)\] Taking \(n \to \infty \) yields the desired result.

    Theorem. Borel-Cantelli Two [001T]

    • December 27, 2024
    • Kellen Kanarios

    If \(\{A_n\}\) is an independent sequence of events and \(\sum _{n}P(A_n)\) diverges, then \(P(\lim \sup _n A_n) = 1\).

    Proof. Consider the complement. By independence, we have \[P(\bigcap _{k = n}^{n + m} A_k^c) = \prod _{k = n}^{n + m}(1 - P(A_k)) \leq \exp (-\sum _{k = n}^{n + m} P(A_k)),\] where the last inequality comes from \(1 - x \leq e^{-x}\). Since the series diverges, taking \(m \to \infty \) yields the desired result.

     In probability, rather than standard convergence, we actually can consider convergence with high probability or almost surely (should come up later). Formally, we care about measuring the probability of the event \(\{\omega \mid \lim _n X_n(\omega ) \to X(\omega )\}\). The first question we must ask is whether this set is measurable. They do not really cover this in the textbook, but it is an important trick that comes up a lot.
    Proof. We consider the complement. Note that the sequence of random variables not converging can be written as \(\forall \epsilon , \exists n\) such that \(|X_n(\omega ) - X(\omega )| \geq \epsilon \). For a fixed \(\epsilon \), this is the \(\lim \sup \) of the sequence \(A_n \coloneqq [|X_n - X| \geq \epsilon ]\). We know that \(\lim \sup \) of measurable sequence is measurable because it is a countable intersection of countable unions. Therefore, the last thing we need is to show for all \(\epsilon \). This can be done just using rationals because \(\mathbb {Q}\) is dense in \(\mathbb {R}\). Then we have a countable union of countable sets as desired.

    Definition. Expected Value [0029]

    • December 28, 2024

    A Simple Random Variable of the form \(X = \sum _{i} x_i I_{A_i}\) is assigned expected value as \[\mathbb {E}[X] = \mathbb {E}\left [\sum _i x_i I_{A_i} \right ] = \sum _i x_i P(A_i)\] More generally, if \(f(X) = \sum _i f(x_i) I_{A_i}\) then \[\mathbb {E}\left [f(X)\right ] = \sum _i f(x_i) P(A_i) = \sum _x f(x) P[X = x]\] A useful identity is as follows: \[\mathbb {E}[X] = \int _0^{\infty } P[X \geq x] \mathrm {d}x\]
