    Definition. Vector Processing Unit [OGCRN]

    • May 22, 2026
    • Kellen Kanarios

    A core component of the TPU architecture responsible for elementwise operations.

    • 2D SIMD execution, where each of 1024 sites execute same program in lockstep.
    • Each site contains 4 ALUs, enabling 4096 FLOPs.
    • Fed a 8x128 Vector Register in one cycle.
