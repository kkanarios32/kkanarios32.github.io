Definition. Huffman Code [005W]

  • March 25, 2025
  • Kellen Kanarios

Given \(x_1, \ldots x_m\) proceed as follows:

  1. Take two \(x_i, x_j\) with smallest \(p(x_i), p(x_j)\) append 0 and 1 to their respective codes.
  2. Combine them and treat them as a new value with \(p(x_i') = p(x_i) + p(x_j)\)
  3. Repeat

    Theorem. Optimality of Huffman Code [0060]

    • March 25, 2025
    • Kellen Kanarios

    Let \(X \sim P_X\) be an \(\mathcal {X}\)-valued random variable. Define \[L_H = \mathbb {E}_{X \sim P_X}[\ell _H(x)]\] and \[L^*(P_X) = \min _{\ell _1, \ldots , \ell _m} \sum _{i = 1}^{m} P_X(x_i) \ell _i.\] Then \(L_H = L^*(P_X)\).

    Remark. SFE satisfies \(L_H < H(X) + \frac {2}{n}\) i.e. it achieves the optimal rate as \(n \to \infty \). Huffman coding achieves optimality for all finite values \(n\).
