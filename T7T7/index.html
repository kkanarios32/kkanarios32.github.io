Definition. Sufficient Statistics [T7T7]

  • June 4, 2026

A statistic \(T(X)\) is sufficient for \(\theta \) if the conditional distribution of the data \(X\) given \(T(X) = t\) does not depend on \(\theta \). Intuitively: if someone hands you \(T(X)\), they could regenerate a fresh dataset with the same distribution without knowing \(\theta \) — so \(T\) lost nothing about \(\theta \).

Example. Bernoulli. \(p(\mathbf {x}; \theta ) = \theta ^{\sum _{i} x_i}(1 - \theta )^{n - \sum _{i} x_i}\). The distribution only depends on \(\mathbf {x}\) through \(\sum _{i} x_i\), so this is a sufficient statistic.
