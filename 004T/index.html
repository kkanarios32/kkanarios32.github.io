Theorem. Convergence of GD with backtracking [004T]

  • February 3, 2025
  • Kellen Kanarios

Let the function \(f(\pmb {x})\) be \(\mu \)-strongly convex and \(L\)-smooth. With backtracking linesearch, the GD method converges with \[f(\pmb {x}_k)-f(\pmb {x}_\star )\ \le \ \left (1-\min \left \{2c_1\mu ,\dfrac {2\alpha c_1\mu }{L}\right \}\right )^k(f(\pmb {x}_0)-f(\pmb {x}_\star ))\] where \(\pmb {x}_\star \) is the unique minimizer.

    Definition. Exact Linesearch [004Q]

    • February 3, 2025
    • Kellen Kanarios

    For \[x_{k+1}\ =\ x_{k}\ -\ \tau _{k}\cdot \nabla f(x_{k})\] Choose step size by solving \[\tau _{k}\;=\;\arg \operatorname *{min}_{t\geq 0}\;f({\mathbf {x}}_{k}-t\cdot \nabla f({\mathbf {x}}_{k}))\]

    Remark. Way to computationally expensive to use in practice.

    Definition. Armijo Condition [004R]

    • February 3, 2025
    • Kellen Kanarios

    For some \(c_1 \in (0,1)\), \[f( \mathbf {x}_k + \tau _k \cdot \mathbf {d}_k) < f( \mathbf {x}_k) + c_1 \cdot \tau _k \cdot \mathbf {d}_{k}^{\top } \nabla f(\mathbf {x}_k)\]

    Definition. Wolfe condition [004S]

    • February 3, 2025
    • Kellen Kanarios

    For some \(0 < c_1 < c_2 < 1\), \[f(\mathbf {x}_{k}+\tau _{k}\cdot \mathbf {d}_{k})\ <\ f(\mathbf {x}_{k})+c_{1}\cdot \tau _{k}\cdot \mathbf {d}_{k}^{\intercal }\nabla f(\mathbf {x}_{k})\] \[d_{k}^{\top }\nabla f({\mathbf {x}}_{k}+\tau _{k}{\mathbf {d}}_{k})\ \geq \ c_{2}{\mathbf {d}}_{k}^{\top }\nabla f({\mathbf {x}}_{k})\]

    Remark. Wolfe condition requires evaluating gradient of function each update to \(\tau _k\). However, Armijo Condition only requires evaluating gradient once. Therefore, it is primarily used in practice.
    Remark. Backtracking is not used in optimizing DNNs because even function evaluation is very expensive.
