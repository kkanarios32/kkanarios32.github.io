1. Interpreting the role of a single neuron

  • September 10, 2025
  • Kellen Kanarios

This is done in Network dissection

Idea 1: find neuron with maximum activation for subsets of image. Need subset dataset.

Problem: How do you do this for all of the neurons in your network? This is not scalable!

Approach 1: use segmentation network to segment images and use these as subsets. This requires the concepts to be detectabl by a segmentation model.

This is done in MILAN

Approach 2: train a "captioner" model i.e. an LSTM to generate captions for parts of images. Can also get labels using clips as done in CLIP-dissect

Problem with these approaches is they are one-shot i.e. we train the model to produce labels and use those in one-shot. We instead want a "loop" that allows us to refine labeler. Namely

  1. Train model to label something like "human faces"
  2. Find sensitive neuron to images labeled "human faces" by the labeler.
  3. Remove the neuron and see if the model becomes unable to detect human faces.
  4. If this isn't the case then we need to refine the labeler.

How can we automate this process?

2. Automating scientific explorations of neural networks

  • September 10, 2025
  • Kellen Kanarios

We want to be able to

  1. Analyze any model.
  2. Answer any interpretability question.

Observation: LLMs can draw via code see Sparks of AGI.

ViperGPT: Given an image reasoning task, an LLM can write code to solve the image reasoning task

Example. 

  • September 10, 2025
  • Kellen Kanarios

Given a picture of kids with some amount of muffins, answer the question

How many muffins can each kid have for it to be fair?

The LLM writes code to identify children / muffins in the image, counting them, and dividing number of muffins by number of kids.

MAIA API:

  • takes image, language description along with model and specific neuron i.e. "neuron 0 layer 4" and it will get the activation for the subset of the image focusing on the language description.
  • Can also take a dataset, or generate new images to test hypothesis.
  • Can go image to text

End to end user can ask "what function does neuron 122 perform" and MAIA will be able to answer.

How do we know the description is correct? One approach is "synthetic neurons". Provide the value of the neuron as the confidence score from SAM for a given language concept description.

Another approach: take the description given by MAIA, then generate images from the prompts along with "negatives" that have nothing to do with the label of MAIA. Then look at the activation value for each of the examples.

3. Auditing and editing neural networks based on interpretations

  • September 10, 2025
  • Kellen Kanarios

For classifiers, can check the bias in the logit neurons. For one example, found beer detection class had a bias toward the gender of the person in the image i.e. low probability for females with beer glass.

Can identify and remove biased neurons. Re-train only selective neurons on biased dataset and can match performance of classifier trained on unbiased dataset.
